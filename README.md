# si5-dsl-teamb
Ce projet vise à implémenter un DSL (Domain-Specific Language) pour la création de divers scripts Arduino afin de répondre aux cas d'utilisation définis dans le lab. 

## Languages utilisés 
* DSL externe : LSP Languim
* DSL interne : Groovy
